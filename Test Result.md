# command

Transmitter command:

```
./spirit1_test -t 50 -i 10000 -e 76800 -p -10
```

-t 50 ,		 Transmitter , package size 50.

-i 10000,	Test run 10000 integrations 

-e 76800,   Air datarate 76800

-p -10,       Max power lever -10dm

Receiver Command:

```
./spirit1_test -r -e 76800 -p -10
```

-r                Receiver 

-e 76800,   Air datarate 76800

-p -10,       Max power lever -10dm
